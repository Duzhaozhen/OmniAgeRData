# OmniAgeData


This is the companion data package for **OmniAgeR**, an R package for calculating aging omic biomarkers.


This repository is currently being prepared for submission to [Bioconductor](https://bioconductor.org/) as an `ExperimentData` package. It contains the necessary pre-trained model weights, clock parameters, and example datasets required by the main `OmniAgeR` software package.
